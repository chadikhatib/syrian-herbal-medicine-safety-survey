## Overview

This repository contains the complete statistical analysis pipeline for the cross-sectional survey (N=300) examining knowledge, attitudes, and practices (KAP) of herbal medicine use among digitally connected Syrian adults, including diaspora communities. The study evaluates readiness for an Arabic-language mHealth application informed by a 67-plant safety database.
